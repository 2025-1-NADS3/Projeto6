
# Testes de Transferência Pix - Projeto Simulado

Este pacote simula a funcionalidade de transferência Pix e contém testes unitários e de integração usando `pytest`.

## Estrutura

- `codigo_fonte/conta_bancaria.py` → classe com a lógica simulada da conta
- `testes/test_pix_unitarios.py` → 4 testes unitários
- `testes/test_pix_integracao.py` → 2 testes de integração

## Como executar

1. Instale o pytest, se necessário:
   pip install pytest

2. Vá até a pasta onde extraiu este projeto e rode:
   pytest testes/

Você verá o resultado dos testes no terminal.
