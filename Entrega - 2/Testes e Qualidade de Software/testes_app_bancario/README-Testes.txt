PASSO A PASSO PARA EXECUTAR OS TESTES

1. Abra o projeto no Android Studio ou em outro ambiente Java.
2. Copie as pastas 'teste', 'modelo' e 'relatorios' para dentro do projeto (caso ainda não estejam).
3. Certifique-se de ter JUnit 4 no classpath.
4. Execute os testes clicando com o botão direito nos arquivos de teste e escolhendo "Run".
5. Para o teste de carga:
   - Rode o arquivo SimulacaoCarga.java como aplicação Java.
   - O resultado estará no arquivo relatorios/resultado_carga.txt.
